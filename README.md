# Система классификации кредитоспособности клиентов банка

Современная система, основанная на знаниях, предназначенная для оценки кредитоспособности физических лиц.

Проект реализует интеллектуальный подход к классификации клиентов на основе экспертных правил и многомерного анализа клиентских данных.

## Основные возможности

- **Редактор базы знаний** — удобный интерфейс для экспертов по добавлению и модификации классов кредитоспособности, признаков и правил
- **Интеллектуальный решатель** — определяет класс кредитоспособности клиента («Высокая», «Средняя», «Низкая», «Невозможная») методом опровержения гипотез
- **Подсистема объяснений** — предоставляет понятные обоснования принятого решения с указанием противоречащих признаков
- **Ввод и валидация исходных данных** — поддержка 13 ключевых признаков (доход, кредитная история, стаж, нагрузка, имущество и др.)
- **Проверка полноты и целостности знаний** — автоматическое выявление пробелов в базе знаний
